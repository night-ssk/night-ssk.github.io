#### 1、基本思想框架
在写回溯问题之前一定要画出树形结构，将树形题目分类再套如下模版：
```c
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }
	//这里可以做剪枝操作
	if(不可能运行的地方) return;
	
    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```
做题目前要清楚的知道终止条件是什么！

#### 3、去重
- 40.组合总和II / 78.子集
对于这种可以对源数组进行排序的可以先排序用（used数组/index）i > index则表示是当前子序列的去重。
false代表了当前的数字已经用过了，其实主要是true代表垂直遍历的时候是可以重复用元素的。主要是要保证同层元素不可以重复。
```c
        if (i > 0 && nums[i] == nums[i - 1] && used[i - 1] == false)
            continue;
		used[i] = true;
		backtracking(nums, i + 1, used);
		used[i] = false;
```

#### 5、括号生成
这个形式的回溯并非树形结构因此不能完全的套用模版
数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。
示例 1：
输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]

示例 2：
输入：n = 1
输出：["()"]
这题目的关键点在于想要输入一个右括号必须有左括号的存在才可以。
